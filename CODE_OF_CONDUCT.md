# Code of Conduct

This project follows the
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
in spirit. The short version:

- **Be kind.** Disagreement on technique is fine; disrespect of people is not.
- **Be specific.** "This is bad" without a fix is noise. Either propose a
  patch or open an issue with a reproducible test.
- **Be honest.** Don't claim a result you didn't verify. Every assertion in
  this codebase is meant to be falsifiable (`verify`, `expect`, `assert`,
  `exit 0`, `must` — same vocabulary Synaptic uses).
- **Respect the kill-file.** Anyone running this loop on their own machine
  should be able to stop it with one command. Don't ship changes that make
  `autopilot off` ineffective.

Conflicts: open an issue or email `support@ersimulator.com`.
